# Hakobu

Universal client for using FTP/SFTP

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'hakobu'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install hakobu

## Usage

Run in you project directory:

    $ hakobu
